# site-template2

## Feature

- DOCTYPE HTML5 Format
- include jQuery 2.x latest
- include Bootstrap latest
- include FontAwesome latest
- use gulp
  - use gulp-sass
  - use gulp-webserver
  - use gulp-postcss
  - use gulp-sourcemaps
  - use autoprefixer-core

## Usage

Install node module.

```
$ mpm install
```

Start local Web Server, Watching scss dir, Compiling SCSS to CSS and apply PostCSS.

```
$ gulp
```
